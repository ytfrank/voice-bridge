### 本轮覆盖
1. Web 模式 App（`http://localhost:3002`）是否能正常加载
2. 测试页（`http://localhost:3001/static/test.html`）文件上传流程
3. 短/中/长音频场景下的识别结果与耗时
4. BFF 健康检查页面
5. 截图、录屏、报告产物是否完整

- 摄像头/麦克风权限交互
- 中文语音专项准确率
- 多用户并发压测

## 三、测试场景与结果

### 场景1：Web 模式 App 加载
- 地址：`http://localhost:3002`
- 结果：**❌ 未通过**
- 发现：页面级错误 `Cannot use 'import.meta' outside a module`
- 影响：当前 Web 模式 App 不能作为“接近真实用户流程”的可用测试入口直接通过验收
- 证据：
  - `tests/artifacts/v17/webapp_01_loaded.png`
  - 录屏包含该步骤：`tests/artifacts/v17/video/v1.7_web_evidence.mp4`

### 场景2：短音频文件（21s Musk）
- 文件：`tests/fixtures/audio/en_short_21s_musk.wav`
- 结果：**✅ 通过**
- 页面流程：选择文件 → 点击“开始识别 + 翻译” → 返回英文原文 + 中文翻译
- 耗时：**2819ms**
- 结论：准确率与延迟均可接受
- 证据：
  - `tests/artifacts/v17/short_01_page.png`
  - `tests/artifacts/v17/short_02_selected.png`
  - `tests/artifacts/v17/short_03_result.png`

### 场景3：中等长度音频（30s Climate）
- 文件：`tests/fixtures/audio/en_medium_30s_climate.wav`
- 结果：**✅ 通过**
- 耗时：**3529ms**
- 结论：识别文本完整，翻译正常
- 证据：
  - `tests/artifacts/v17/medium_01_page.png`
  - `tests/artifacts/v17/medium_02_selected.png`
  - `tests/artifacts/v17/medium_03_result.png`

### 场景4：长音频文件（3min AI）
- 文件：`tests/fixtures/audio/en_medium_3min_ai.wav`
- 结果：**✅ 通过**
- 耗时：**8684ms**
- 结论：长音频切片链路可用，页面可返回完整结果
- 证据：
  - `tests/artifacts/v17/long_01_page.png`
  - `tests/artifacts/v17/long_02_selected.png`
  - `tests/artifacts/v17/long_03_result.png`

### 场景5：BFF 健康检查
- 结果：**✅ 通过**
- 证据：`tests/artifacts/v17/health_01_result.png`

## 四、当前风险判断

### 已验证可成立
- **测试页方式**可以作为无 Expo Go 开发者弹窗干扰的替代测试入口
- 短/中/长音频文件在该测试页下都能成功完成识别与翻译
- 能稳定产出可留痕的页面级证据

### 当前不能直接说通过的点
- **Web 模式 App 本身未通过页面加载验证**
- 因此，如果目标是“App Web 入口本身可直接作为接近用户真实流程的验收入口”，**当前证据不支持通过**
